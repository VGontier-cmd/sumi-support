# Sumi · Support

> Apprends à lire les hiragana et katakana japonais. Quiz adaptatif, répétition espacée, vocabulaire de base. Design minimaliste, sans publicité.

[![Privacy](https://img.shields.io/badge/data-100%25%20local-2ea44f)](PRIVACY.md)

Ce dépôt est le canal de support officiel de l'application **Sumi**. Le code source de l'app n'est pas open-source ; ce repo existe uniquement pour :

- Recueillir les **bugs** et le **feedback** des utilisateurs
- Héberger la **politique de confidentialité** publique requise par les stores

## Tu as un problème ou une suggestion ?

- **Bug ou comportement inattendu** : [ouvre une issue](https://github.com/VGontier-cmd/sumi-support/issues/new?template=bug.yml)
- **Suggestion ou retour d'expérience** : [ouvre une issue](https://github.com/VGontier-cmd/sumi-support/issues/new?template=feedback.yml)

Les issues sont relues régulièrement. Précise au minimum ton modèle de téléphone et la version de l'app (visible tout en bas des Réglages) pour aider au diagnostic.

## Privacy policy

L'app ne collecte, ne transmet, ni ne stocke à distance aucune donnée personnelle. Détails complets : [PRIVACY.md](PRIVACY.md).

## Pour les curieux : ce que fait l'app

- **107 hiragana + 107 katakana** avec mnémoniques, audio (TTS), et étymologie
- **Quiz adaptatif** qui choisit le type de question (lecture / écriture / tracé / écoute) selon ta maîtrise du kana
- **Répétition espacée** (algorithme SM-2 simplifié) pour optimiser la révision
- **Stroke order animé** via les vraies données [KanjiVG](https://kanjivg.tagaini.net) (CC-BY-SA)
- **30+ achievements** dont une dizaine cachées
- **Streaks** avec un système de "shield" hebdomadaire
- **Lecture en contexte** : à la fin de chaque chapitre, mots japonais lisibles avec les kana appris
- **50 mots de vocabulaire** courants
