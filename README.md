# Sumi · Support

> Learn to read Japanese hiragana and katakana. Adaptive quizzes, spaced repetition, vocabulary basics. Minimalist design, no ads.

[![Privacy](https://img.shields.io/badge/data-100%25%20local-2ea44f)](PRIVACY.md)

This repository is the official support channel for the **Sumi** mobile app. The app's source code is not open-source; this repo exists only to:

- Collect **bug reports** and **feedback** from users
- Host the public **privacy policy** required by the app stores

## Got an issue or a suggestion?

- **Bug or unexpected behaviour** — [open an issue](https://github.com/VGontier-cmd/sumi-support/issues/new?template=bug.yml)
- **Suggestion or feedback** — [open an issue](https://github.com/VGontier-cmd/sumi-support/issues/new?template=feedback.yml)

Issues are reviewed regularly. Please include at least your phone model and the app version (visible at the bottom of the in-app Settings) to help with diagnosis.

## Privacy policy

The app does not collect, transmit, or remotely store any personal data. Full details: [PRIVACY.md](PRIVACY.md).

## What the app does

- **107 hiragana + 107 katakana** with mnemonics, audio (TTS), and etymology
- **Adaptive quiz** that picks the question type (reading / writing / drawing / listening) based on your mastery of each kana
- **Spaced repetition** (simplified SM-2 algorithm) for efficient review
- **Animated stroke order** powered by the canonical [KanjiVG](https://kanjivg.tagaini.net) dataset (CC-BY-SA)
- **30+ achievements**, including hidden ones
- **Streaks** with a weekly "shield" system
- **In-context reading** — at the end of each chapter, real Japanese words you can now read
- **50 essential vocabulary words** to apply what you've learned
